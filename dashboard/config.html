Configuración del servidor de correo

La configuración SMTP se gestiona desde el archivo .env en el servidor. Agrega o edita estas variables:

# Servidor SMTP
SMTP_HOST=mail.tuempresa.com
SMTP_PORT=587
SMTP_USER=soc@tuempresa.com
SMTP_PASSWORD=tu_password
SMTP_FROM=SOC Leviathan <soc@tuempresa.com>
SMTP_TLS=true
# URL del portal SOC (para links en los mails)
SOC_URL=10.1.108.82:8888
# Credenciales del panel de config
CONFIG_USER=admin
CONFIG_PASSWORD=tu_password_seguro
